# 🧬 DNA Sequence Analyzer

An interactive web application for biological DNA sequence analysis, built with Python and Streamlit. Designed to bridge biology domain knowledge with modern data analytics and visualization.

## 🔍 What It Does

This app allows users to input or fetch a DNA sequence and instantly perform a range of bioinformatics analyses — all through a clean, browser-based interface.

## ✨ Features

- **ORF Detection** — Identifies Open Reading Frames within a given DNA sequence
- **Nucleotide Frequency Analysis** — Visualizes the distribution of A, T, G, C bases
- **GC Content Calculation** — Computes GC percentage, a key indicator in molecular biology
- **NCBI GenBank Integration** — Fetches real genomic data live from the NCBI database using accession numbers
- **Interactive Charts** — Data visualizations powered by Python plotting libraries

## 🚀 How to Run Locally

1. Clone the repository
```bash
https://github.com/premkrishnarnair2100-ally/dna-analyzer
cd dna-analyzer
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the app
```bash
streamlit run app.py
```

4. Open your browser at `http://localhost:8501`
